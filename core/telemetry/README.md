# Telemetry Module

The telemetry module will normalize measurements from CentroidOS runtimes.

Planned metric families:

- reflex latency
- deliberation latency
- narrative reconciliation delay
- action correction timing
- memory recall consistency
- identity drift
- contradiction detection
- node synchronization latency
- failover continuity
- stability-weighted planning

Telemetry must not include private memory content or credentials. Public
telemetry should describe measurable runtime behavior only.
